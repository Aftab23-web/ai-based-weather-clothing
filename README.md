# 🤖 AIStyleRec
## AI-Powered Style Recommendations & Weekly Planning

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-3.1.1-green.svg)](https://flask.palletsprojects.com/)
[![AI/ML](https://img.shields.io/badge/AI%2FML-scikit--learn-orange.svg)](https://scikit-learn.org/)

> **AIStyleRec** is an intelligent clothing recommendation system that combines AI/ML algorithms with real-time weather data to provide personalized outfit suggestions and weekly style planning.

## 🚀 Features

### 🤖 **AI-Powered Recommendations**
- Machine learning algorithms analyze weather conditions
- Personalized suggestions based on user profile (age, gender, preferences)
- Smart outfit matching for any weather scenario

### 🌍 **Real-Time Weather Integration**
- Live weather data from global cities
- Temperature, humidity, wind speed, and pressure analysis
- 7-day weather forecasting

### 👔 **Personalized Styling**
- **Casual & Formal** style options
- Age-appropriate recommendations (Child, Teen, Adult, Senior)
- Gender-specific outfit suggestions

### 📅 **Weekly Planning**
- 7-day outfit planning for travel or daily use
- Email delivery of weekly recommendations
- PDF download for offline access

### 🛒 **Shopping Integration**
- Direct links to purchase recommended items
- Curated products from trusted retailers
- Visual product previews with images

### 📧 **Email Notifications**
- Detailed recommendations sent to your email
- Weekly planner delivery
- Safety tips and weather alerts

### 📊 **User Dashboard**
- Personal recommendation history
- Admin analytics (for admin users)
- Usage statistics and trends

## 🛠️ Technology Stack

### **Backend**
- **Flask** - Web framework

## 📱 How It Works

### 1. **User Registration**
- Create account with personal details
- Specify age, gender, and style preferences
- Secure password-based authentication

### 2. **Weather Analysis**
- Enter your city name
- System fetches real-time weather data
- AI analyzes temperature, humidity, and conditions

### 3. **AI Recommendation**
- Machine learning model processes weather + user profile
- Generates personalized outfit suggestions
- Provides shopping links for each recommended item

### 4. **Weekly Planning**
- Enter travel destination
- Get 7-day weather forecast
- Receive daily outfit recommendations
- Download PDF planner
